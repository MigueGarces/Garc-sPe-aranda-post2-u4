## Descripción
Programa en lenguaje ensamblador NASM que implementa macros con 
parámetros, bucles con LOOP y condicionales con CMP/Jcc.

## Prerrequisitos
- DOSBox
- NASM instalado en Windows
- Archivos .asm en la misma carpeta

## Compilación
nasm -f bin programa2.asm -o programa2.com -l programa2.lst

## Ejecución
Abrir DOSBox y ejecutar:
- MOUNT C C:\DOSBOX
- C:
- CD TRABAJO
- P2.COM
